## 関数名について
名前付き関数（greet）
```javascript
function greet() {
  console.log("Hello!");
}
```
`function() { ... } `部分が **無名関数**
```javascript
const greet = function() {
  console.log("Hello!");
};
```
無名関数は、一時的に関数を使いたいときや、コールバックとして渡すときによく使われる。

## JSON

`JavaScript Object Notation`の略で、データを表現するための軽量なデータ交換フォーマット。
人間の目にも読み書きしやすく、機械からも解析しやすいという特徴から、Web APIなどで広く使われている。

### 形式
```json
{
  "name": "た"
}
```

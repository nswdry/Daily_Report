# わかったこと💡

##  ブロックとは
- Rubyにおけるブロックは、メソッドに渡すことができる**一時的な処理のまとまり**のこと。
- 複数行の処理には `do...end`を、1行で終わる簡単な処理には`{}`を使うのが一般的。

##  メソッドの作り方
```ruby
def メソッド名
  # 実行したい処理
end
```
- メソッドは作成されただけでは中の処理は実行されず、呼び出すことで実行される。
- メソッドで最後に実行した結果が戻り値になる。

### 戻り値
メソッドや関数が処理を終えた後に返す結果のこと。

```ruby
def add(a, b)
  return a + b
end

result = add(3, 5) # addメソッドを呼び出す
puts result        # 8が戻り値
```

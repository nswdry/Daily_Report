## Vimの扱い方について

| コマンド |      効果      |
| :------: | :------------: |
|    0     |  行頭まで移動  |
|    $     |  行末まで移動  |
|    dd    |   1行を削除    |
|    p     |    貼り付け    |
|    w     | 次の単語に移動 |
|    x     |   1文字削除    |
|    dw    |   1単語削除    |

## promiseチェーン

- promiseをメソッドチェーンで繋ぐこと

### メソッドチェーンとは

- メソッドを呼び出した返り値に対してさらにメソッド呼び出しをするパターンのこと。
- 配列で頻出するパターンの一つ。
- promiseの`.then()`もメソッドチェーンにあたる。
- 1行に複数の処理を書くことができる。

```javascript
# メソッドチェーンの例`.concat()`

const array = ["a"].concat("b").concat("c");
console.log(array); // => ["a", "b", "c"]
```

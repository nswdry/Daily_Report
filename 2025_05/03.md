## promiseを使用するメリット
昔のJavaScriptでは、複数の非同期処理を連続して実行すると**コールバック地獄**に陥ってしまいましたが、**promise**の使用によって、ネストが深くなってみづらいという問題を解消できるようになりました。

```javascript
const promise = doSomething();
const promise2 = promise.then(successCallback, failureCallback);
```

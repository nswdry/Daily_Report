 ## ES Modulesの適用方法

- `package.json`に以下のコードを追加する
```javascript
{
  "type": "module"
}
```
もしくは、
- ファイルの拡張子を`.mjs`に変更することで`package.json`の設定がなくてもES Modulesが適用される。

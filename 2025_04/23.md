# わかったこと💡

## アロー関数
- 構文
```javascript
// 通常の関数
function add(a, b) {
  return a + b;
}

// アロー関数
const add = (a, b) => a + b;
```

アロー関数を使うと、通常の関数を簡潔に書くことが出来る！

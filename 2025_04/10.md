## コールバック形式とコールバック関数の違い

### コールバック形式

コールバック関数を使う「コードの書き方・設計スタイル」のこと。

### コールバック関数

時間が経った時や、ユーザがボタンを押したときなど後から呼ばれる関数のこと。
処理が終わったときに呼び出す「関数」そのものを指す。

```javascript
() => { ... }
```

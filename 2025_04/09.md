## 作業ブランチにmainブランチの内容を取り込む

```javascript
$ git checkout main // mainブランチへ移動

$ git pull origin main // リモートの変更を取得

$ git checkout <作業ブランチ名> // 作業ブランチへ戻る

$ git merge origin main // mainブランチの変更を作業ブランチに取り込む
```
